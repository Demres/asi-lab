StudentLog przechowuję informacje o studentach:
* imiona
* nazwiska
* zainteresowania
* zdjęcia

Dodatkowe Gemy:
* faker - tworzenie studnetów
* devise - logowanie do aplikacji
* paperclip - możliwość dodania zdjęcia
* bootstrap-sas - do tworzenia wyglądu
